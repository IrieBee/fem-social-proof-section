## Built with

  * HTML
  * CSS
  * Flexbox
  * Grid

### What I learned

* Use align-self: flex-end and flex-start to make 'stairs' from flexboxes.
* Use function calc() for margin-left on rating box.
* To fix jumping around quote containers -> height: 100%
* background: vs background-image:
